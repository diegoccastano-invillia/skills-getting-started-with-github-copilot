# Começando com o GitHub Copilot

_Comece a usar o GitHub Copilot em menos de uma hora._

## Boas-vindas

- **Para quem é**: Pessoas desenvolvedoras de qualquer nível de experiência que queiram acelerar seu fluxo de trabalho com código.
- **O que você vai aprender**: As diferentes formas de interagir com o Copilot para explicar, escrever, planejar e desenvolver código.
- **O que você vai construir**: Você vai guiar o Copilot para atualizar o site de atividades extracurriculares da Mergington High School.
- **Pré-requisitos**:
  - Exercício do Skills: [Introduction to GitHub](https://github.com/skills/introduction-to-github)
  - Familiaridade com o [VS Code](https://code.visualstudio.com/)
  - Princípios básicos de programação
- **Quanto tempo leva**: Este exercício leva menos de uma hora para ser concluído.

Neste exercício, você vai:

1. Usar um Codespace pré-configurado para rodar o VS Code no seu navegador.
1. Aprender diferentes opções de interação para desenvolver e planejar com o GitHub Copilot.
1. Usar o Copilot para resumir e revisar seu pull request.

### Como começar este exercício

Basta copiar o exercício para a sua conta e dar à sua Octocat favorita (Mona) **cerca de 20 segundos** para preparar a primeira lição. Em seguida, **atualize a página**.
